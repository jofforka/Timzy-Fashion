# Timzy Fashion OS v8

## Firebase setup

1. Enable Email/Password in Firebase Authentication.
2. Create the admin user.
3. Publish `firestore.rules`.
4. Publish `storage.rules`.
5. Upload the full v8 bundle to GitHub Pages.
6. Open `admin/index.html`.

## Important files

- `assets/js/config.js`
- `assets/js/firebase.js`
- `assets/js/app.js`
- `admin/index.html`
- `admin/admin.js`
- `admin/admin.css`

The admin supports product editing, image replacement, multiple image uploads, Instagram Reel/Post links, sales, expenses, and storefront updates through Firestore.
