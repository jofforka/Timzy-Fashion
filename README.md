# Timzy Fashion v7 Public Release
Updated Home, Training, Contact, navigation, CSS and contact form behavior.

## Product Instagram Video

Add the Instagram Reel or post URL to each product in `data/products.json`:

```json
"instagramVideoUrl": "https://www.instagram.com/reel/POST_ID/"
```

The video button is hidden when the field is empty. Clicking the button opens an embedded Instagram viewer without leaving the website.
